# School CI server.Typescript

Добавил типы для:

* Компонентов
* Редьюсеров и экшенов
* Ручек сервера
* Своих классов и функций
* Моделей с базы данных

## Запуск

```bash
git clone https://github.com/doyouwannatea/shri-node-hw
cd ./shri-node-hw
git checkout types
npm run install
npm run build
npm start
```
